Overview

This notebook demonstrates zero-shot image segmentation using SAM1 (Segment Anything Model, Meta AI). SAM1 is a foundation model trained on over 1 billion masks across 11 million images, enabling powerful generalization to novel domains — including electron microscopy images — without any task-specific fine-tuning.

Key concepts

  • Zero-shot segmentation: SAM1 requires no labeled electrochemical training data. Its broad pretraining allows it to delineate object boundaries in SEM images out of the box.

  • Two usage modes:

    • Automatic mask generation (SamAutomaticMaskGenerator): segments all detectable objects in an image, returning a list of candidate masks sorted by area.

    • Prompt-based segmentation (SamPredictor): segments a specific region guided by a user-provided point, box, or mask prompt — useful for targeting individual particles.

  • Backbone: ViT-H (Vision Transformer, huge), pretrained by Meta AI on SA-1B.

Workflow

  1. Load the ViT-H SAM1 checkpoint (downloaded automatically if absent).

  2. Run automatic mask generation on a sample SEM image.

  3. Post-process masks: sort by area, combine into a single binary image, and inspect top-ranked regions.

  4. Run prompt-based segmentation using a point click to isolate a specific particle.

Mask Post-Processing: Sorting, Combining, and Top-K Selection

SAM1’s automatic generator returns a list of masks, each annotated with area, predicted IoU score, and bounding box. This step:

  • Sorts all masks by pixel area in descending order, so the largest detected regions appear first.

  • Combines all masks into a single binary image where every detected object is marked as foreground — useful as a complete particle/background segmentation map.

  • Selects the top-2 largest masks into a separate binary image, isolating the dominant structural regions (e.g., the largest electrochemical particles or electrode layers).

This post-processing converts the raw list of SAM predictions into actionable segmentation maps for downstream analysis.

[ ]:

# Sort masks by area in descending order
sorted_masks = sorted(masks, key=lambda x: x['area'], reverse=True)
sorted_masks_top2 = sorted_masks[:2] # Get the top 2 masks

# Make all masks as a single binary image
binary_mask = np.zeros_like(image[:, :, 0])
for mask in masks:
    binary_mask[mask["segmentation"]] = 1

# Create a binary mask for the top 2 masks
binary_mask_top2 = np.zeros_like(image[:, :, 0])
for mask in sorted_masks_top2:
    binary_mask_top2[mask["segmentation"]] = 1

logging.info(f"Generated {len(masks)} masks, with the top 2 masks having areas: {[mask['area'] for mask in sorted_masks_top2]}")

Prompt-Based Segmentation with SamPredictor

While automatic mask generation identifies all detectable objects, prompt-based segmentation targets a specific region using a user-supplied coordinate. The SamPredictor interface encodes the image once and then efficiently processes any number of point, box, or mask prompts.

Here, a single foreground point at pixel coordinates (400, 300) is passed as the prompt. SAM1 returns a single binary mask delineating the object that contains that point. This approach is ideal for interactively isolating a specific particle or structural feature in an electrochemical micrograph without needing to sift through all automatic predictions.

[ ]:

# Use SamPredictor for prompt-based segmentation
predictor = SamPredictor(sam)
predictor.set_image(image)

# Define a point prompt (x, y coordinates) and label (1 = foreground, 0 = background)
input_point = np.array([[400, 300]])  # Click at (400, 300)
input_label = np.array([1])  # Label as foreground

# Generate mask
masks, scores, logits = predictor.predict(
    point_coords=input_point,
    point_labels=input_label,
    multimask_output=False
)

# Plot result
fig, ax = plt.subplots(1, 2, figsize=(12, 5))
ax[0].imshow(image)
ax[0].plot(input_point[:, 0], input_point[:, 1], 'r*', markersize=20, label='Prompt')
ax[0].legend()
ax[0].set_title("Input Image with Point Prompt")
ax[0].axis("off")

ax[1].imshow(masks[0], cmap="gray")
ax[1].set_title("Segmented Mask")
ax[1].axis("off")
plt.show()
